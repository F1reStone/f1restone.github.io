## 概述

FireStone 火石（以下简称「我们」）非常重视您的隐私，特此制定本《隐私政策》以便您了解当您访问和使用本网站时，我们如何收集、使用、存储、共享和保护相关信息。

继续使用本网站，即表示您已阅读并理解本隐私政策。

## 我们收集的信息

我们可能收集以下类型的信息：

### 1. 您主动提供的信息

当您通过评论或其他方式与我们联系时，您可能会向我们提供：

- 昵称
- 电子邮件地址
- 消息内容
- 您主动提交的其他信息

### 2. 自动收集的信息

当您访问本网站时，我们可能会自动收集与访问和设备相关的信息，例如：

- 页面访问记录
- 浏览器类型与版本
- 设备类型与操作系统
- 访问时间
- 来源页面
- 大致地区信息
- 交互事件和页面停留数据

### 3. Cookie 与类似技术

本网站使用了 Cookie 与类似技术，用于：

- 维持网站基础功能
- 记住您的偏好设置
- 分析访问情况
- 改善内容和使用体验

我们使用的第三方 Cookie 包括但不限于：

|  第三方 Cookie*[^*1]  |  目的  |
|  ----------  |  ----------  |
|  百度统计  |  用于收集网站访问情况 |
|  Google Analytics  |  用于收集网站访问情况 |
|  Cloudflare Web Analytics  |  用于收集匿名访问数据并优化网站性能  |
|  Microsoft Clarity  |  用于收集用户行为分析数据  |

您可以通过网站左下角提供的 Cookie 设置，或通过浏览器设置管理 Cookie。请注意，禁用 Cookie 可能会影响网站的部分功能。

## 我们如何使用这些信息

我们可能将收集到的信息用于以下目的：

- 提供和维护网站
- 优化页面性能与内容结构
- 分析访问趋势和使用行为
- 记住您的偏好设置
- 回应您的联系请求
- 防范滥用、攻击和安全风险

## 第三方服务

本网站可能使用第三方服务来提供分析、基础设施、内容分发或安全保护功能。此类服务可能在其服务范围内独立处理部分数据。

常见第三方服务可能包括：

- 网站分析服务
- 内容分发网络（CDN）
- 安全与防护服务
- 嵌入式内容服务

我们使用的第三方服务包括但不限于：

|  第三方服务*[^*1]  |  目的  |
|  ----------  |  ----------  |
|  GitHub Pages  |  用于托管网站 |
|  Cloudflare CDN  |  用于加速网站体验  |
|  Cloudflare Turnstile  |  用于保护网站访问安全  |

关于我们使用的第三方 Cookie，请参见 [「Cookie 与类似技术」部分](#3-cookie-与类似技术)。

## 数据共享

我们不会出售您的个人信息。

在以下情况下，我们可能会共享必要信息：

- 您已明确同意
- 为提供网站功能所必需
- 为保护网站、用户或公众的安全和权益
- 为防止欺诈、滥用或技术攻击

## 数据保存

我们仅在实现本政策所述目的所需的期限内保存相关信息。

当信息不再需要时，我们会采取合理措施删除或匿名化处理。

## 您的选择

您可以：

- 管理或拒绝非必要 Cookie
- 关闭浏览器中的所有 Cookie
- 联系我们，了解与您相关的数据处理情况

请注意，某些功能依赖必要 Cookie 才能正常运行。但这些必要 Cookie 不会收集您的个人信息，我们绝不会用必要 Cookie 来收集您的个人信息。

## 儿童信息

我们不会有意收集未成年人敏感个人信息。

如果您认为未成年人意外向我们提供了信息，请联系我们，我们会尽快处理。

## 安全措施

我们会根据网站的实际情况采取合理的技术和组织措施，以保护信息免受未经授权的访问、泄露、修改或破坏。

但没有任何互联网传输或存储方式能够保证百分之百安全。

## 本政策的变更

我们可能会不时更新本隐私政策，以反映功能、技术、法律或运营上的变化。

更新后的版本会发布在本页面，并注明最新更新日期。

## 联系方式

如果您对本隐私政策有任何问题，请通过本网站的 GitHub Issue 与我们联系。
